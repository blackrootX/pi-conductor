# pi-conductor

A pi agent package providing basic commands.

## Commands

- `/hello` - Says hello
- `/hello <agent-id>` - Prefills the editor with that agent's frontmatter from `.pi/agents`, `~/.pi/agent/agents`, or built-in agents

## Installation

```bash
pi install git:github.com/blackrootX/pi-conductor
```

Or from the repository URL:

```bash
pi install https://github.com/blackrootX/pi-conductor
```
